### 📖 Usage

```bash
python -m src.main <command> [options]
```

#### Commands

| Command | Description |
|---------|-------------|
| `load_ontologies [path]` | Parse, embed, and register ontology files from a directory. Uses the default path from settings if omitted. |

Example:
```bash
python -m src.main load_ontologies ./ontologies
```
